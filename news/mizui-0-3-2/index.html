mizui 0.3.2

Oct 25, 2021
Nov 23, 2021
1 minute
styleuisassreleaseproject

N3NX mizui v0.3.2 brings new changes with this update, repo link here.

More information for this update can be found here

Features

  • core: New typography font classes for many font weights and styles (5fb4edf)

Improvements

  • core: Add strong and em tags to general typography class (ce1895f)
  • core: Optimize display classes for property one-liners (e3a4680)
  • defaults: Integrate serif class into typefaces (a705063)

Refactoring Updates

  • core: Remove obsolete bootstrap code and old tooltip design (9693245)
  • core: Move misc utilities from other files to (e53560b)
  • defaults: Move as to core (d8fc7a3)

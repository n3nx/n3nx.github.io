Code of Conduct

Aug 08, 2021
Dec 20, 2023
1 minute
cocguidelinesfoundation

The N3N Community Code of Conduct weaves together four pillars for a harmonious and thriving environment:

Code

  1. Excellence: Uphold a commitment to continuous improvement and the pursuit of excellence in your contributions.
  2. Growth: Embrace opportunities for personal and collective growth, fostering an environment that supports each member's journey to becoming a better version of themselves.
  3. Kindness: Interact with fellow members with kindness, respect, and empathy, fostering an inclusive and supportive community.
  4. Innovation: Encourage and engage in collaborative innovation, where diverse minds come together to create solutions that make a positive impact.

Adherence to N3N Philosophy

This code aligns with the N3N's coding philosophy to create a holistic foundation for a thriving community.

Contact

This code is a guide, not a rigid set of rules. For concerns, connect with us at coc@n3n.org.
